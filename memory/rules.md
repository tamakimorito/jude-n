# ジュデーン 開発ルール

## 基本ルール
1. 単一HTMLファイルで構成（サーバー不要）
2. 複数の文字コード対応（UTF-8, Shift_JIS, CP932）
3. コピー機能はタブ区切りでExcel/スプレッドシート互換

## デザインルール
1. ダークテーマ（黒背景）
2. ウルトラ怪獣をイメージ（赤・オレンジ系アクセント）
3. フッタには「©タマシステム 2025」を必ず表示

## コード規約
1. JavaScriptはモダンなES6+構文を使用
2. CSSはVanilla CSSを使用
3. レスポンシブ対応

## バージョン管理
- バージョンは0.01刻みで更新
- 変更内容はchangelog.mdに記録
