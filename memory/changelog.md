# ジュデーン バージョン履歴

## v0.01 (2025-12-25)
- 初回リリース
- Zoom Phone CSVログ解析機能実装
- 不在着信抽出ロジック実装
- 折り返し済みチェック機能実装
- クリップボードコピー機能
- ダークテーマ（ウルトラ怪獣風デザイン）

## v0.02 (2025-12-25)
- 電話番号の表示形式をハイフン区切り（000-0000-0000）に変更

## v0.03 (2025-12-25)
- 科学的記法（8.19049E+11形式）の電話番号解析を改善
- toFixed(0)とBigIntを使用して精度問題を軽減
- 注意: CSVがExcelで保存された際に既に精度が失われている場合は復元不可能

## v0.04 (2025-12-25)
- **Excel(.xlsx)ファイルのサポートを追加**
- SheetJSライブラリを使用してExcelファイルから正確な電話番号を読み取り
- CSVで精度が失われている場合でも、XLSXファイルなら正確なデータを取得可能

## v0.05 (2025-12-25)
- 不要な警告表示を削除
- 科学的記法（8.15E+11形式）でも折り返し判定は正しく動作
- Zoomが出力するCSVフォーマットに対応
